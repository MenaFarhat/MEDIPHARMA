# MEDIPHARMA

MediPharma is a high-performance mobile application developed with **Flutter** and **Dart**, designed to streamline healthcare management. The app provides an intuitive interface for users to manage appointments, access medical information, and interact with healthcare services efficiently.

---

## Features
- Schedule and manage healthcare appointments
- Access detailed medical information
- User-friendly and responsive UI/UX
- Built following clean architecture and best development practices
- Cross-platform mobile application (Android & iOS)

---

## Tools & Technologies
- **Flutter**
- **Dart**
- **Figma** (for UI/UX design)

---

## Getting Started

### Prerequisites
- Flutter SDK installed
- Android Studio or Xcode (for mobile emulators)
- Git installed

### Installation
```bash
# Clone the repository
git clone https://github.com/MinaFarhat/MEDIPHARMA.git
cd MEDIPHARMA

# Install dependencies
flutter pub get

# Run the app on an emulator or connected device
flutter run
